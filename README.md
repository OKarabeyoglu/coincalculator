# coincalculator

### REQUIREMENTS
* Java 8
* Maven
* PostgreSQL

## RUN THE PROJECT

- Clone the project
- Configure PostgreSQL configurations in application.properties
- Create coin database and coin schema in PostgreSQL
- Build the project
- Run the application
- http://localhost:8080/swagger-ui.html

### With DOCKER

- Build the project
- Run commands below:
```
docker build . 
```
```
docker-compose up -d 
```
